INDIGO IAM repository

This is the INDIGO IAM repository website.

Docker images

INDIGO IAM Docker images are available on Dockerhub under the indigoiam organization. The recommended way to deploy INDIGO IAM is from the provided Docker images.

Images are provided for the:

  • IAM login service application
  • IAM test client application

Packages

Starting with release 1.5.0, IAM packages are signed. Get the IAM release key from here.

Stable repositories

These repositories hold stable, certified releases suitable for use in production.

  • RHEL 9 (repo file)
  • RHEL 7 (repo file)
